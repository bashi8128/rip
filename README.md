# rip
Rust implementation of PJT

```
Usage:
  rip [options] [<file>]
  rip (-h|--help)

Options:
  -p <order>        output BinTree
  -s SUBST          substitute specified node and output BinTree
  -u                output BinTree with a deduplication
  -r REMOVE         remove specified node
  -h,--help         output this help
```
